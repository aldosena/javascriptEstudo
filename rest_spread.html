Exemplo 1: Rest

O operador rest é usado para agrupar argumentos em um array.

Repare que a variável titulo recebe a string, mas todos os outros parametros são salvos na variável numeros, que passa a ser uma string

Se o último argumento nomeado de uma função tiver prefixo com ..., ele irá se tornar um array

spread

Serve para pegar um array e "espalhar" os dados.
Usando onde, por exemplo, um array não pode ser lido
No exemplo abaixo, array não seria permitido 

const numeros = [1,2,3];
console.log(Math.max(...numeros));
