Objetivo

Objetos são estruturas de dados que armazenam propriedades e valores
No array, uso []
No objeto, uso {}
Exemplo de uso 

	let person = {
		name: 'aldo',
		age: 30,
		weight: 80
		};

Outro exemplo

	// criar o array
	let personagem = {
		nome: 'Batman',
		idade: '30',
		pais: 'usa',
		cores: ['black', 'gray'],
		caracteristica: {
			forca: 20,
			maria: 5,
			vida: 10,
		},
	}
	// exibir tudo
	console.log(personagem);
	// exibir alguns
	console.log(`${personagem.nome} tem ${personagem.idade} anos.`);
	// exibir o vlaor de forca
	console.log(personagem.caracteristica.forca);
	// exibir a cor principal
	console.log(personagem.cores[0]);

Alterando objetos 

	personagem.nome = 'Superman'; // mudar no nome do personagem
	personagem.caracteristica.forca = 77; // mudando outro dado.
	personagem.cores.push('white'); // acrescentera uma cor
	console.log(personagem);
